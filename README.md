# Lucide Icon Gallery

A simple React application to display and explore the Lucide icon set.

---

## Project Overview

* Built with React.
* UI utilizes Tailwind CSS for styling and responsiveness.

---

## Future Plans

* Implement a **local, offline icon search functionality**.

---

## Setup and Run

1. Clone this repository.
2. Install dependencies: `npm install` or `yarn install`
3. Start the development server: `npm run start` or `yarn start`
